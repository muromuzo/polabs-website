## 핵심 트렌드 요약

### 1. 대담한 타이포그래피 (Bold Typography)
- **초대형 폰트 사이즈** (80px-200px+)
- 굵은 서체 + 가벼운 서체 대비
- **텍스트 애니메이션**: 글자별 등장, 스크롤 기반 변형
- 텍스트가 이미지를 대체하는 히어로 디자인
- **적용 예**: jinsul.kr의 "Chapter" 글자별 애니메이션, 히어로 라인별 슬라이드업

### 2. 다크 모드 & 강렬한 컬러
- 다크 배경 + 네온/골드 강조색
- **컬러풀한 그라디언트** 복귀 (2026)
- 섹션별 배경색 전환
- **적용 예**: jinsul.kr의 #000 → #391109 → #222 → #5f0000 배경 전환

### 3. 스크롤 기반 인터랙션 (Scroll-Driven)
- **GSAP ScrollTrigger** 활용 증가
- 패럴랙스 / 핀 / 스크럽 효과
- 스크롤 진행 인디케이터
- 섹션 진입 시 요소 등장 애니메이션
- **적용 예**: GSAP 커튼 효과, 원형 이미지 스케일업

### 4. 몰입형 경험 (Immersive Experience)
- 풀스크린 섹션
- 마우스 인터랙션 (트레일, 커서 변형)
- 노이즈/그레인 텍스처 오버레이
- 시네마틱 이미지 트랜지션
- **적용 예**: jinsul.kr의 마우스 트레일 캔버스, 노이즈 오버레이

### 5. 마이크로 인터랙션
- 버튼 호버 애니메이션
- 스무스 스크롤
- 로딩 애니메이션
- 커서 커스터마이징
- **적용 예**: 네비게이션 메뉴 항목 라인 애니메이션, 화살표 bounce

### 6. 모바일 퍼스트 & 반응형
- 모바일 우선 설계
- 터치 친화적 인터페이스
- 세로 스크롤 최적화
- 간결한 모바일 네비게이션

### 7. AI & 개인화 (2026 신규)
- AI 기반 콘텐츠 추천
- 동적 레이아웃 조정
- 챗봇/상담 위젯 고도화

## 한국 기업/병원 홈페이지 특화 트렌드

### 디자인
- **원페이지 구조** 선호 (스크롤 기반 스토리텔링)
- **다크 톤** 프리미엄 이미지 (컨설팅, 의료 분야)
- 대형 타이포 + 키 비주얼 조합
- 세리프 + 산세리프 혼합 사용

### CTA / 전환
- **카카오톡 실시간 상담** (우측 고정)
- **전화 상담 신청** (고정 버튼)
- **상담 폼** (이름, 연락처, 선택항목)
- 누적 상담 수 카운터 (신뢰 요소)

### 콘텐츠 구성
- Chapter 기반 스토리텔링 (1→2→3→...)
- 실적/숫자 강조 (5년 150곳, 5000개 입지 등)
- 포트폴리오 이미지 슬라이더
- 법무법인 제휴 등 신뢰 요소
