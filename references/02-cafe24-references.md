## WEBBASE - 반응형원페이지 01

### HTML 구조 패턴
```
├── Header (고정, 투명 → 스크롤 시 배경)
│   ├── 로고 (좌측)
│   ├── 네비게이션 (중앙) - ABOUT / SERVICE / PORTFOLIO / CLIENTS / CONTACT
│   └── CTA 전화번호 (우측, border pill 버튼)
├── Hero Section (풀스크린 슬라이더)
│   ├── 배경 이미지 (블러/오버레이)
│   ├── 소제목 (영문, 작은 크기)
│   ├── 대제목 (초대형 볼드 타이포)
│   └── 슬라이드 인디케이터 (01 — 02 — 03 — 04)
├── About Section
│   ├── 2컬럼 (이미지 좌 + 텍스트 우)
│   └── 대형 영문 타이틀 + 본문
├── Company Profile (다운로드 CTA)
├── Service Section
│   ├── 3x2 그리드 카드
│   └── 서비스명 + 설명 텍스트
├── Counter Section (다크 배경)
│   └── 4개 숫자 카운트업 (CLIENTS / PHOTOS / AWARDS 등)
├── Portfolio Section
│   └── 이미지 그리드/갤러리
├── Clients Section
│   └── 로고 나열
├── Contact Section
│   └── 문의 폼
└── Footer
```

### CSS 특징

#### 1. 풀스크린 히어로
- `height: 100vh` + `background-size: cover`
- 반투명 다크 오버레이 (`rgba(0,0,0,0.4)`)
- 타이포그래피: 100px+ 볼드 산세리프

#### 2. 고정 헤더
- `position: fixed` + 투명 배경
- 스크롤 시 배경색 전환 (scroll 이벤트)
- 우측 CTA: `border: 1px solid #fff` pill 버튼

#### 3. 서비스 그리드
- `display: grid; grid-template-columns: repeat(3, 1fr)`
- 호버 시 배경색/텍스트색 전환
- 각 카드: 제목(대문자) + 설명 텍스트

#### 4. 카운터 섹션
- 다크 배경 (`#222` 또는 `#333`)
- 4열 가로 배치
- 스크롤 진입 시 숫자 카운트업 애니메이션

#### 5. 반응형 패턴
- 데스크탑: 1200px 컨테이너
- 태블릿: 2열 그리드
- 모바일: 1열 스택, 햄버거 메뉴

### JavaScript 특징
- jQuery 기반
- 스크롤 이벤트: 헤더 배경 전환
- 카운터: waypoint/intersection 기반 카운트업
- 슬라이더: Swiper 또는 자체 구현
- 스무스 스크롤: anchor link

### HTML 구조 패턴
```

│   ├── 소제목 (영문)

│   ├── 부제목
│   └── 슬라이드 번호 인디케이터 (01-04 진행 바)

├── About 섹션 (이미지 + 텍스트)
├── Product 섹션
├── Contact 섹션
```

### CSS 특징
- 다크 헤더 (불투명 `#333`)
- 히어로: 이미지 슬라이더 + 텍스트 페이드인
- 3열 카드: `flex` 또는 `grid`
- 소셜 고정 버튼 (카카오톡, 인스타, 유튜브, 블로그)
- top 버튼 (우측 하단 고정)

## 공통 디자인 패턴 (카페24 템플릿)

### 레이아웃
1. **풀스크린 히어로** - 거의 모든 템플릿 공통
2. **고정 헤더** - 투명 → 스크롤 시 배경 전환
3. **섹션 기반 구성** - Hero → About → Service → Portfolio → Contact → Footer
4. **우측 하단 고정 버튼** - 소셜 링크, Top 버튼

### 타이포그래피
- 대형 제목: 60-100px+
- 영문 소제목 + 한글 대제목 조합
- 볼드 + 가벼운 무게 대비

### 인터랙션
- 스크롤 기반 등장 애니메이션 (AOS 패턴)
- 카운트업 숫자
- 이미지 슬라이더 (Swiper)
- 호버 효과 (카드, 버튼)

### CTA 패턴
- 고정 카카오톡/전화 버튼
- 상담 신청 폼
- pill 형태 CTA 버튼
